## 재귀 호출시 주의해야 할 점
- 재귀호출을 하면 n이 50이상일 때
  - 시간 초과가 나거나 런타임 에러가 난다.
  - 런타임 에러가 나는 이유는, 재귀 호출을 할 수 있는 횟수가 정해져 있고, 횟수는 넘어 재귀 호출을 하면 런타임 에러를 내도록 설계되어있기 때문이다.

### 코드를 어떻게 바꿔야 시간초과 혹은 런타임 에러를 막을 수 있을까?
- 재귀 호출 대신 for문을 사용해 피보나치 수를 순서대로 구해본다. (DP)
- 피보나치 수열 예시
~~~python
def fibonacci(num):
    a, b = 0, 1
    for i in range(num):
        a, b = b, a+b
    return a
~~~
